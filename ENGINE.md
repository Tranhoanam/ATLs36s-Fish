# Native Android Canvas 2D Engine

## Overview
The Native Android Canvas 2D engine is designed to leverage the powerful features of Android's built-in graphics library, providing an efficient platform for rendering 2D graphics with speed and flexibility.

## 2D Sprites & Cyber Grunge Graphics
Utilize 2D sprites that can be easily animated and manipulated, allowing for a dynamic gaming experience. The aesthetic can draw inspiration from Cyber Grunge graphics, which combine a gritty, futuristic aesthetic with vivid colors and textures.

## System Requirements
- **Minimum RAM:** 512 MB
  - The engine is optimized to run on devices with at least 512MB RAM, ensuring decent performance on most Android devices.

## Circular Button Mechanism
The engine supports a circular button mechanism, allowing for more intuitive input from users. These buttons can be used for various in-game actions, such as movement or activating abilities, ensuring a seamless interactive experience.

## Fish HP Bar System
An integrated HP bar system allows for the representation of the health points of fish characters. This visual indicator aids players in tracking the health status and making strategic decisions during gameplay.
